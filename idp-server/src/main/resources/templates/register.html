Validación inicial de identidad

El registro no habilita el acceso inmediatamente. Primero se valida el correo, luego el usuario inicia sesión y configura TOTP.

  • Contraseña protegida con hash BCrypt.
  • Correo enviado por SMTP real.
  • Token de verificación con expiración.
  • El login se bloquea si el correo no está verificado.
  • El acceso completo requiere TOTP.
